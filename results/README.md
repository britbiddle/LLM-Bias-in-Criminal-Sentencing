## Re-running

Open `notebooks/ablation_study.ipynb` and run top-to-bottom. The run saves to `results/` (not to `data/` / `figures/` subfolders — the notebook doesn't know about them yet). If you want new outputs to land in the subfolders automatically, change cell 4:

```python
RESULTS_DIR = ROOT / 'results'
DATA_DIR    = RESULTS_DIR / 'data'
FIGURES_DIR = RESULTS_DIR / 'figures'
for d in (DATA_DIR, FIGURES_DIR): d.mkdir(parents=True, exist_ok=True)
```

…then s/RESULTS_DIR/FIGURES_DIR/ in every `plt.savefig(...)` and s/RESULTS_DIR/DATA_DIR/ in every `to_parquet` / `to_csv` / `to_json` call. Until then, after a re-run you can rerun the `mv` commands documented in the commit that set this folder up.
